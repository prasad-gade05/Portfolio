Prasad Gade Agent Endpoint

This page is the public agent entry point for prasadgade.dev. It exists so AI agents and developers can find the structured, static files that describe the portfolio.

What this site is

Prasad Gade's portfolio is a personal website, not just a resume. It presents technical work, projects, skills, and contact information, while also showing personality, hobbies, favorite media, and creative interactions such as the paper playground, Minecraft skin viewer, multiple themes, and small easter eggs.

Machine-readable files

  • A2A Agent Card
  • agents.json discovery file
  • API catalog
  • Agent skills index
  • OpenAPI specification
  • llms.txt
  • llms-full.txt
  • XML sitemap
  • robots.txt

Static API endpoints

  • /api/resume.json - resume data
  • /api/projects.json - projects and technical work
  • /api/skills.json - categorized skills
  • /api/social.json - contact links and profiles
  • /api/about.json - site philosophy, hobbies, and personality

Usage notes

The portfolio is hosted as a static GitHub Pages site. There is no live server-side agent runtime, OAuth flow, or MCP transport server behind this page. The available interface is static discovery plus static JSON data.
